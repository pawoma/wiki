前后端接口规范

HTTP 接口返回格式 (Scheme)

接口状态码 (HTTP Code): => 200

接口输出 (Response): JSON => {errorCode, msg, data, success}

{
  "errorCode":200000, // 错误状态
  "msg":"OK", // 返回提示语  
  "data":[]||{}, // 返回数组或对象          
  "success":true// 请求是否成功   ture-成功 false-失败      
}

errorCode 字段状态码定义

  • 状态码为Number类型
  • 200000-请求成功
  • 430000-未登陆

其他

  • 接口输出必须是规范的 JSON 字符串, 拒绝手动拼接 json 字符串返回，请参考 http://json.org/
